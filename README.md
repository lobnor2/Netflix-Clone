# Netflix-Clone

## Live Demo

https://netflix-clone-pearl-six.vercel.app/

## Run Locally

Clone the project
```bash
  git clone https://github.com/lobnor2/Netflix-Clone
```

Go to the project directory

```bash
  cd Netflix-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
